Natural Earth is one of the best public domain data sets and now you can instantly use it for your mapping projects by using the prerendered vector or raster tiles.

Check out the GitHub repository for usage guides and more information.

Vector Tiles

The Natural Earth Vector Tiles support the Mapbox Vector Tile Spec and consist of carefully curated layers from the cultural and physical vector data from Natural Earth. Style the tiles with Mapbox Studio and display them with Mapbox GL JS.

Raster Tiles

The different Raster tile sets are traditional WebP raster tiles made from the Natural Earth Raster TIF files. You can mix and match them with the vector tiles or use them in other web clients.
